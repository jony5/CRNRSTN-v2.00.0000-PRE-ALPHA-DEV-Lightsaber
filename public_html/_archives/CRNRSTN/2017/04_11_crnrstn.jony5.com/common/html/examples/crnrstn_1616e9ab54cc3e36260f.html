The environmentals :: class object is constructed with its own instantiation of the http_manager :: class object called oHTTP_MGR. In the below example of a classic and common client-server interaction, the http_manager :: class object facilitates the handling of search data sent to the server via an AJAX request handler that is wrapped around a users' search input field.

After determining on line 14 that (1) $_GET data has been provided in the specified parameter through the invocation of issetParam() and that (2) the length of data is sufficient to allow for the successful execution of the code within the logic statement (via extractData()), the web services layer is then invoked through a custom (not part of CRNRSTN) user :: class objects' suggestSearchResults() method (line 15) to retrieve the available search suggestions from the database.
