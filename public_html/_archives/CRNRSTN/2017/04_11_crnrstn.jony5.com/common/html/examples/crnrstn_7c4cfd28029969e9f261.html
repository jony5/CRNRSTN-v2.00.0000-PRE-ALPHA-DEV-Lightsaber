In the below example of a classic and common client-server interaction, the http_manager :: class object facilitates the handling of data sent to the server via an AJAX request handler that is wrapped around a users' search input field.

On line 19 below, issetParam() is invoked in order to determine if data is available within the specified parameter to warrant a round trip to the web services end point.
