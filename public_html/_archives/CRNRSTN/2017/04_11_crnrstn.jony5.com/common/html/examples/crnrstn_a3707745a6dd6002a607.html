The CRNRSTN Suite :: stands on a stack of class definitions. The CRNRSTN configuration file (_crnrstn.config.inc.php) comes preconfigured to include all of the necessary class definitions. Here is an example of said class definition includes along with the instantiation of the crnrstn :: class object.
