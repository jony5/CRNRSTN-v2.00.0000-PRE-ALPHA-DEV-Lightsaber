To access the environmentally specific parameters configured into the CRNRSTN Suite :: simply invoke the getEnvParam() method of the environmentals :: class object.

In the following excerpt from the CRNRSTN Suite :: configuration file, we have the configuration of the DOCUMENT_ROOT parameter (line 6).
