Each environment can be configured to handle notices passed through the CRNRSTN Suite :: logging class object. There are currently four (4) modes through which notices can be handled. If ":EMAIL" or "FILE" is specified as the $loggingMode, the appropriate $endPointDetails needs to be specified (either comma delimited email addresses or path+filename respectively).
