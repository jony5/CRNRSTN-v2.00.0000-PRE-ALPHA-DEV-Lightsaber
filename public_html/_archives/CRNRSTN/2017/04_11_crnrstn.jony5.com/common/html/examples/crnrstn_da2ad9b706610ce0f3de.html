The CRNRSTN Suite :: stands on a stack of class definitions. The CRNRSTN configuration file (_crnrstn.config.inc.php) comes preconfigured to include all of the necessary class definitions. Here is an example of said class definitions along with the instantiation of the crnrstn :: class object.

To see a complete example of the CRNRSTN Suite :: configuration file, click here. For a more tutorial-ish presentation of the progression through the same, click here for documentation and examples covering the next few lines.
