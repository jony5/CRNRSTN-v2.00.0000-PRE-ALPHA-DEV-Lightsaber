The environmentals :: class object is constructed with its own instantiation of the http_manager :: class object. In the below example of a classic and common client-server interaction, the http_manager :: class object facilitates the handling of data sent to the server via form submission (not shown) for user authentication.

After determining that $_POST data has indeed been provided (on line 13) through the invocation of issetHTTP(), the extractData() method of the http_manager :: class object is then called to return a parameter ('postid') which both identifies the source of the HTTP request and determines the data handling requirements. This particular request is processed through a web services layer powered by NuSOAP and the soap_manager :: class of the CRNRSTN Suite ::.
