If-Then Lesson 3

This lesson is similar to the previous two lessons except it will demonstrate that the Then and End instructions are optional if only a single statement is to be executed when the condition is true.

The format of the If instruction is:

        : If condition
        :     statement 1
        : next program instruction
        : etc.
        :


The If instruction works by first evaluating the condition to see if it is true or false. If the condition is true, the statement immediately following the If is executed. If the condition is false, the statement immediately following the If is ignored, and execution of the program resumes at the next program instruction.

The following example shows an If condition with only a single statement to be executed if the condition is true. Notice that the optional Then and End statements have been left out! However, if more than one statement is to be executed if the condition is true, then those statements must be enclosed within a Then and End block.

:ClLCD
:Input "ENTER YOUR AGE  ", A
:If A  >=  16
:Disp "YOU CAN DRIVE"
:Disp "  "
:Disp "FINISHED"
  ENTER YOUR AGE   16
   YOU CAN DRIVE
 
   FINISHED

 

   ENTER YOUR AGE  14
   
   FINISHED
