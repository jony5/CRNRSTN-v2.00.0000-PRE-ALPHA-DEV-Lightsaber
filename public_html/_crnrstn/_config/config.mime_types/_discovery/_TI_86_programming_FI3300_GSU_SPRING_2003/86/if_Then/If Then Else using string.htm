If-Then Lesson 5

The If -Then - Else instruction will execute one block of instructions if the condition is true, and it executes a different block of instructions if the condition is false

New Instructions Introduced:

  • If - Then - Else


The format of the If - Then - Else instruction is:

        : If condition
        : Then
        :     statement 1
        :     statement 2
        :     etc.
        : Else
        :    statement 1
        :    statement 2
        :    etc.
        :End

Note: If, Then, Else, and End are all separate statements and you must select each one individually from the [PRGM] [CTL] menu


The If - Then - Else instruction works by first evaluating the condition to see if it is true or false. If the condition is true, the first block of statements between Then and Else are executed. If the condition is false, the second block of statements between Else and End are executed.

In the following example, the user will enter their class (9-12) and the program will use a string variable to print whether they are an underclassman or an upperclassman.

:ClLCD
:Disp "ENTER YOUR CLASS"
:Disp "( 9 - 12 )"
:Prompt  C
:If  C   >=  11
:Then
:"UPPERCLASSMAN" --> GRADE
:Else
:"UNDERCLASSMAN" --> GRADE
:End
:Disp "  "
:Disp "YOU'RE  AN ", GRADE
   ENTER YOUR CLASS
   ( 9 - 12 )
   C =  10
 

 

   YOU'RE  AN
   UNDERCLASSMAN
