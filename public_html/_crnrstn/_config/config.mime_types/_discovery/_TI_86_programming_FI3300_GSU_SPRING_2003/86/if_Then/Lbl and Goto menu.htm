If-Then Lesson 7

This lesson  will demonstrate how the Lbl and Goto instructions may be used with the If instruction to control the flow of a program.

New Instructions Introduced:

  • Lbl - This instruction assigns a label to a program command. The label can be one to eight characters long, starting with a letter.

  • Goto - When the program encounters a Goto instruction, the flow of the program immediately transfers to the line where the specified label is located.


The format of the instructions are:

        :Lbl label

        :Goto   label


Note: The Lbl and Goto instructions are found in the [PRGM] [ CTL ] menu.

Programs are often written which contain several parts and the user is given the option of which part to run. In this example, the Lbl and Goto instructions are combined with the If instruction to set up a menu to allow the user to control which part of the program is executed.

:ClLCD
:Disp "PLEASE CHOOSE: "
:Disp "(1) PART  A"
:Disp "(2) PART  B"
:Disp "(3) PART  C"
:Input "ENTER 1, 2, OR 3: ", C
:Disp " "
:If  C == 1
:Goto PARTA
:If  C == 2
:Goto PARTB
:If  C == 3
:Goto PARTC
:
:Lbl PARTA
:Disp "THIS IS PART  A"
:Goto DONE
:
:Lbl PARTB
:Disp "THIS IS PART  B"
:Goto DONE
:
:Lbl PARTC
:Disp "THIS IS PART  C"
:
:Lbl DONE
:Disp " "
:Disp "FINISHED"
  PLEASE CHOOSE:
   (1) PART   A
   (2) PART   B
   (3) PART   C
   ENTER 1, 2, OR 3:  1

   THIS IS PART  A

   FINISHED

  PLEASE CHOOSE:
   (1) PART   A
   (2) PART   B
   (3) PART   C
   ENTER 1, 2, OR 3:  2

   THIS IS PART  B

   FINISHED

   PLEASE CHOOSE:
   (1) PART   A
   (2) PART   B
   (3) PART   C
   ENTER 1, 2, OR 3:  3

   THIS IS PART  C

   FINISHED
