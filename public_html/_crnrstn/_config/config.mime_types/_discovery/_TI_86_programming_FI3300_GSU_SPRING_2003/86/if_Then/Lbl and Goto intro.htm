If-Then Lesson 6

This lesson  will demonstrate how the Lbl and Goto instructions may be used with the If instruction to control the flow of a program.

New Instructions Introduced:

  • Lbl - This instruction assigns a label to a program command. The label can be one to eight characters long, starting with a letter.

  • Goto - When the program encounters a Goto instruction, the flow of the program immediately transfers to the line where the specified label is located.


The format of the instructions are:

        :Lbl label

        :Goto   label


Note: The Lbl and Goto instructions are found in the [PRGM] [ CTL ] menu.

This example will demonstrated how the Lbl and Goto instructions may be used to allow the user the option of re-running the program. The user enters a number and is then given the opportunity to re-run the program by entering a 1 or a 2.

:Lbl  A
:ClLCD
:Input "ENTER A NUMBER   ", N
:Disp "YOUR NUMBER WAS:" , N
:Disp " RUN AGAIN ? "  
:Disp "(1) YES"
:Disp "(2) NO"
:Input "ENTER 1 OR 2: ",C
:If C   ==  1
:Goto A
:Disp " "
:Disp "FINISHED"
