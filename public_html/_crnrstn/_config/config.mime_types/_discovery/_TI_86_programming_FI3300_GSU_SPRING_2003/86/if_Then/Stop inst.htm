If-Then Lesson 9

This lesson is similar to If-Then Lesson 7 except it will demonstrate how the Stop instruction may be used to bring a program to a halt at a particular point.

New Instructions Introduced:

  • Stop - This instruction terminates the execution of a program and returns to the home screen.



Note: The Stop instruction is found in the [PRGM] [ CTL ] menu.

Programs are often written which contain several parts and the user is given the option of which part to run. In this example, the Lbl and Goto instructions are combined with the If instruction to set up a menu to allow the user to control which part of the program is executed.

A Stop instruction may be included at the end of each program option to end the program after that part has executed.

:ClLCD
:Disp "PLEASE CHOOSE: "
:Disp "(1) PART  A"
:Disp "(2) PART  B"
:Disp "(3) PART  C"
:Input "ENTER 1, 2, OR 3: ", C
:Disp " "
:If  C  ==  1
:Goto PARTA
:If  C  ==   2
:Goto PARTB
:If  C  ==   3
:Goto PARTC
:
:Lbl PARTA
:Disp "THIS IS PART  A"
:Stop
:
:Lbl PARTB
:Disp "THIS IS PART  B"
:Stop
:
:Lbl PARTC
:Disp "THIS IS PART  C"
:Stop
