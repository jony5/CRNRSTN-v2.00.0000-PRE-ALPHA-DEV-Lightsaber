For Loop Lesson 4

Although it may be used in several ways, the For loop is generally used to repeat a block of statements a fixed number of times.

The format of the For loop is:

        : For ( var, initial, final, stepsize )
        : statement 1
        : statement 2
        : etc.
        : End

where:   
        var - This is the loop control variable.
        initial - This is the initial value of the loop control variable.
        final - This is the final value of the loop control variable.
        stepsize - This is the incremental value.

Note: The For instruction is found in the [PRGM] [CTL] menu.

The For loop works by first setting the loop control variable equal to the initial value. The loop then repeats all the statements between For and End, incrementing the control variable with each pass through the loop. The amount of the increment is given by the stepsize value. The looping action finally stops when the value of the control variable exceeds the final value. The execution of the program resumes at the first statement following the End statement.

Note: The stepsize is optional and if omitted is assumed to be + 1

 

This example will demonstrate how the stepsize can be used to run the loop backwards by using a negative stepsize.

:ClLCD
:For ( M , 5 , 1 , -1  )
:Disp  M
:End
                                               5
                                               4
                                               3
                                               2
                                               1
