While and Repeat Loop Lesson 3

The While loop is another looping structure that repeats a block of statements within a loop.

The format of the While loop is:

        : While condition
        :
        : statements to be executed if condition is true
        :
        : End


Note: The While instruction is found in the [PRGM] [CTL] menu.

The While loop works by first evaluating the condition to see if it is true or false. While the condition is true, the block of statements inside the loop are executed repeatedly. When the condition becomes false, the looping action stops and the program transfers execution to the first statement following the End statement.

The While loop is generally used in situations where you may not know in advance how many times the loop will be executed.

 

This lesson will demonstrate how a While loop may be used to force the user to repeat an Input instruction until a correct value is entered. This is particularly useful when writing programs which will serve as tutorials.

The first Input statement in this program is called a "priming" Input statement because it "primes the loop". The Input statement inside the loop is repeated until the user enters the correct answer.

:ClLCD
:Input " 5 + 2 = ", A
:While A  =/=  7
:Disp "WRONG"
:Disp "TRY AGAIN"
:Input " 5 + 2 = ", A
:End
:Disp "CORRECT"
   5 + 2 = 8
   WRONG
   TRY AGAIN
   5 + 2 = 7
   CORRECT
