While and Repeat Loop Lesson 2

The While loop is another looping structure that repeats a block of statements within a loop.

The format of the While loop is:

        : While condition
        :
        : statements to be executed if condition is true
        :
        : End


Note: The While instruction is found in the [PRGM] [CTL] menu.

The While loop works by first evaluating the condition to see if it is true or false. While the condition is true, the block of statements inside the loop are executed repeatedly. When the condition becomes false, the looping action stops and the program transfers execution to the first statement following the End statement.

The While loop is generally used in situations where you may not know in advance how many times the loop will be executed.

 

This lesson will demonstrate how a While loop can become an infinite loop if it is not properly constructed.

When using While loops, you must include a statement within the loop that changes the value of the loop control variable. If this statement is omitted, the program may enter an infinite loop which would run forever without stopping. Using the previous lesson, observe the effect of accidentally omitting the statement : C + 1 --> C

Notice that if there is no statement to change the value of the loop control variable ( C ), the condition will always be true and the loop will never end !

Note: To interrupt a program while it is running, hit the [ on ] key !

:1 --> C
:While C <= 5
:Disp C
:End
