I/O Lesson 14

This lesson  will demonstrate how the output may be displayed in a horizontal list form by using braces {  } found in the [LIST] menu.

When the output from a program is enclosed within braces in the Disp instruction, the values are displayed horizontally across the screen within braces.

Note: The braces are found in the [2nd][LIST] menu.

:ClLCD
: 4 --> A
: 6 --> B
: 8 --> C
:Disp A , B , C
:Disp { A , B , C }
                                                4
                                                6
                                                8
                                  { 4  6  8 }
