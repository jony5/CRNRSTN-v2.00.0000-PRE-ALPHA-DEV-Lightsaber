I/O Lesson 2

This lesson will demonstrate how to print a long message.

 

If your message requires more than one line on the calculator screen, you must use several Disp instructions to display it. This lesson will print the message "THIS SENTENCE REQUIRES MORE THAN ONE LINE" using more than one Disp instruction.

 

The ClLCD and Disp instructions are both located in the [PRGM][I / O] menu.

Tip:  Hit [alpha][alpha] to lock the calculator in the alpha mode.

:ClLCD
:Disp "THIS SENTENCE"
:Disp "REQUIRES MORE"
:Disp "THAN ONE LINE"
  THIS SENTENCE
  REQUIRES MORE
  THAN ONE LINE
