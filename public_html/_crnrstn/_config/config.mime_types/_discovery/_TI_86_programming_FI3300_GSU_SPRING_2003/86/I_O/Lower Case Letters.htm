I/O Lesson 3

This lesson will demonstrate how to print using lowercase letters and how to put blank spaces between lines.

 

First, hit [2nd][alpha][alpha] to lock the calcalculator in the lowercase alpha mode. Then type in your message.

To insert a blank line between lines, use Disp " ". ( use two consecutive " )

 

The ClLCD and Disp instructions are both located in the [PRGM][I / O] menu.

:ClLCD
:Disp "This sentence"
:Disp " "
:Disp "skips one line "
:Disp " "
:Disp "in between"
  This sentence

  skips one line

  in between
