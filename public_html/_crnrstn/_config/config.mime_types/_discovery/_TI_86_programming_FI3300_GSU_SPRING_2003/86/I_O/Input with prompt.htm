I/O Lesson 15

This lesson  will demonstrate how the Input instruction may be used to allow the user to enter values into the calculator and store those values in specified variables. This lesson will include a message to prompt the user.

New Instructions Introduced:

  • Input - This instruction allows the user enter values into the calculator and store those values in the variables specified in the Input instruction. You may include an optional message as a prompt to the user.


The format of the Input instruction is:

       Input  "TEXT" , variable
or    Input  variable


The Input instruction is found in the [PRGM] [ I / O ] menu

The following example demonstrates how a message to prompt the user may be included within an Input instruction:

:ClLCD
:Input "Enter A =  ", A
:Input "Enter B =   ", B
:Disp "A = ", A
:Disp "B = ", B
   Enter A = 8
   Enter B = 3
   A =
                                               8
   B =
                                               3
