I/O Lesson 6

This lesson will demonstrate how to use the Prompt instruction to enter values into the calculator and store those values in variables. One or more variables may be included within
a single Prompt statement. The Prompt instruction displays the variable name, including =? , when executed.

 

New Instructions Introduced:

  • Prompt - Allows the user to enter values for the variables contained in the Prompt instruction.

Note: Variable names must conform to the following rules:

1. From one to eight characters in length.
2. Must begin with a letter.
3. May be upper or lower case. ( TAX and Tax are considered to be different ! )
   ( The TI-86 is case sensitive! )
4. You cannot use TI-86 reserved names as variable names.


The Disp instruction may be used to display the value of a variable on the screen. The
variable should be separated from a short descriptive message by a comma in the
Disp instruction.

The Prompt instruction is located in the [PRGM][I / O] menu.

:ClLCD
:Prompt F
:Prompt S , T
:ClLCD
:Disp "FIRST = ", F
:Disp "SECOND = ", S
:Disp "THIRD = ", T

 
  F =? 4.5
  S =? 6
  T =? 7.38

 

  FIRST =
                                              4.5
  SECOND =
                                                 6
  THIRD =
                                             7.38
