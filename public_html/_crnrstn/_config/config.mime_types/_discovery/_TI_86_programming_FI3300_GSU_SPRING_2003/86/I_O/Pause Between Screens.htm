I/O Lesson 8

This lesson will use the Pause instruction to show how to make the program pause between output screens so the user has time to read the information on each screen. The program will pause at each screen until the user hits the [ENTER] key to advance to the next screen display.

 

New Instructions Introduced:

  • Pause - This instruction tells the program to pause until the user hits the [ENTER] key.


The Pause instruction is located in the [PRGM][CTL] menu.

Tip: Place a ClLCD instruction after each Pause instruction if you want to begin with a clean screen.

:ClLCD
:Disp "THIS IS SCREEN  ONE"
:Outpt( 7 , 1 , "HIT ENTER
TO CONTINUE")
:Pause
:ClLCD
:Disp "THIS IS SCREEN
TWO"
:Outpt( 7 , 1 , "HIT ENTER
TO CONTINUE")
:Pause
:ClLCD
:Disp "THIS IS SCREEN
THREE"
