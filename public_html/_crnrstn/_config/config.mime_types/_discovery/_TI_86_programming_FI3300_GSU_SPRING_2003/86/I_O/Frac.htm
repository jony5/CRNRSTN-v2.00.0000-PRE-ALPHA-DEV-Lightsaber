I/O Lesson 10

This lesson is the same as the previous lesson, however, it will use the >Frac instruction to display the output in fractional form instead of decimal form.

 

New Instructions Introduced:

  • >Frac - When this instruction is placed after the variables name in a Disp instruction, the value will be displayed in fractional form reduced to simplest terms

 

Note: If the number cannot be simplified, or if the denominator is more than four digits, it will be displayed in decimal form.

:ClLCD
:6.375  --> A
:8 / 3 --> B
:Disp "A = ", A >Frac
:Disp "B = ", B >Frac
   A =
                                          51 / 8
   B =
                                            8 / 3
