I/O Lesson 11

This lesson is the same as the previous lesson, however, it will use the round instruction to allow the programmer to control the number of decimal places contained in the output.

 

New Instructions Introduced:

  • round - When this instruction is used in a Disp instruction, it allows the programmer to specify the number of decimal places displayed in the output.

The format of the round instruction is: round ( var name , # of decimals )

 

The round instruction is found in the [2nd][MATH][NUM] menu.

Note: The # of decimals ranges from 0 to 11, and the result is rounded, not truncated.

:ClLCD
: 6.375  --> A
: 8 / 3 --> B
:Disp "A = ", round ( A , 2 )
:Disp "B = ", round ( B , 5 )
   A =
                                            6.38
   B =
                                      2.66667
