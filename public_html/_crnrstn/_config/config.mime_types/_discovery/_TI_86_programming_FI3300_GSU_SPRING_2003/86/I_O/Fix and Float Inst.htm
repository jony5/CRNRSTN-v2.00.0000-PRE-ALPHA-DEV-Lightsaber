I/O Lesson 12

This lesson is the same as the previous lesson, however, it will use the Fix instruction to demonstrate another way the programmer can control the number of decimals displayed in the output of a program.

 

New Instructions Introduced:

  • Fix - When this instruction, including the number of decimal places desired, is inserted at any point in the program, all output will be displayed using the specified number of decimals.

The format of the Fix instruction is: Fix   # of decimals 
( ex: Fix 4 )

Notes:

  1. The number of decimal places ranges from 0 to 11.

  2. The Float instruction may be used following a Fix instruction to revert  the calculator back to the full default value of 12 decimals.

  3. The Fix instruction works with both the Disp and Outpt instructions.

The Fix instruction is inserted in a program by highlighting the Float instruction in the [2nd][MODE] menu. Then use the right arrow key to highlight the number of decimal places desired, then hit [ENTER]. The word Float will automatically change to Fix when it is inserted in your program.

 

Tip: It is always a good idea to include a Float ( with no number attached ) at the end of your program to reset the calculator back to the full 12 decimal mode. If you don't, when the program has finished running, the calculator will remain set to the number of decimals specified by the Fix instruction in the program !

:ClLCD
:Fix  4
:6.375  --> A
:8 / 3 --> B
:Disp "A = ", A
:Outpt ( 3 , 1 ,"B =   ")
:Outpt ( 3 , 5 , B )
:Float
   A =
                                        6.3750
   B = 2.6667
