I/O Lesson 5

This lesson will demonstrate how to use the Outpt instruction to control the placement
of information on the screen.

 

New Instructions Introduced:

  • Outpt - Used to display information at a specific row and column locations
                 on the screen.

The format of the Outpt instruction is:  Outpt ( row#, col#, value )

This instruction will display the value listed beginning at the specified row and column.

 

The Outpt instruction is located in the [PRGM][I / O] menu.

:ClLCD
:Outpt( 2 , 12 , "HELLO" )
:Outpt( 4 , 6 , "WORLD" )

                      HELLO

           WORLD
