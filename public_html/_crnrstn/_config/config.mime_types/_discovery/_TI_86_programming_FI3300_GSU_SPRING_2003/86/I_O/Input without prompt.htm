I/O Lesson 16

This lesson  is similar to the previous lesson, however, it will demonstrate how a separate Disp instruction may be used immediately before an Input instruction to act as a prompt for the user.

New Instructions Introduced:

  • Input - This instruction allows the user enter values into the calculator and store those values in the variables specified in the Input instruction. You may include an optional message as a prompt to the user.


The format of the Input instruction is:

       Input  "TEXT" , variable
or    Input  variable


The Input instruction is found in the [PRGM] [ I / O ] menu

Note: Notice in this example the prompt to the user appears on a separate line. Compare this to the previous lesson in which the prompt was contained within the Input instruction.

:ClLCD
:Disp "Enter A"
:Input A
:Disp "Enter B"
:Input   B
:ClLCD
:Disp "A = ", A
:Disp "B = ", B
   Enter A
   ? 8
   Enter B
   ? 3
 
   A =
                                               8
   B =
                                               3
