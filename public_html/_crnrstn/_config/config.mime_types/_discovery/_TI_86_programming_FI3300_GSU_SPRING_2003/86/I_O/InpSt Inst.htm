I/O Lesson 17

This lesson  will demonstrate how the InpSt instruction may be used to enter string values into the calculator instead of numeric values. These string values may then be displayed on the output screen.

New Instructions Introduced:

  • InpSt - This instruction allows the user enter string values into the calculator and store those values in the variables specified in the InpSt instruction. You may include an optional message as a prompt to the user.


The format of the InpSt instruction is:

       InpSt  "TEXT" , variable
or    InpSt  variable


The InpSt instruction is found in the [PRGM] [ I / O ] menu

Note: In this example, two short strings , F and L, are added together into a single larger string and the resulting string is stored in a new string variable FULLNAME.

:ClLCD
:InpSt "Enter first name:  ", F
:Disp "Enter last name:"
:InpSt   L
:ClLCD
:Disp F
:Disp L
:F + "  " + L --> FULLNAME
:Outpt ( 4, 1, FULLNAME )
