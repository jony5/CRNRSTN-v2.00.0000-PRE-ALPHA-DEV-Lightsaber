I/O Lesson 9

This lesson will demonstrate how to use the [STO>] key to store values in variables within the program.

 

The [STO>] key is located on the lower left side of the keyboard.

After values have been stored in variables using the [STO>] key, they may be displayed using the Disp or Outpt instruction.

Note: In all program examples on this website, the [STO>] key will be represented by the symbol -->

:ClLCD
:6.375  --> A
:8 / 3 --> B
:Disp "A = ", A
:Disp "B = ", B
   A =
                                          6.375
   B =
                          2.66666666667
