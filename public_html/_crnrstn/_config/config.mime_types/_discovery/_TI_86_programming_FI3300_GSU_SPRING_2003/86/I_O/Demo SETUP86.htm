I/O Lesson 19

This lesson will demonstrate how the TI-86 SETUP86 program may be called as the first line of any program to reset all calculator internal variables to their default values before the main program executes.

Sometimes you will run a program and the program will produce unexpected results or perhaps it will not run at all. One reason this may occur is that the normal default setting in the calculator have been changed by some previous operation. The setup program listed on this page will reset all the values of the internal calculator variables to their default settings, and this will help insure that your programs will produce the expected results. I strongly suggest that you enter this SETUP86 program into your calculator and run it before executing your own programs, or whenever you want to return the calculator to its default settings.

Tip: It is possible to call this setup program from your own programs! If you include "SETUP86" as the first line of each of your programs, this TI-86 setup program will be called and executed before each program runs.

Note: When you run this program the output will simply say "Done", and you may think nothing has happened. Don't worry! If you check, you will find that all the internal variables have been reset to their default settings.

PROGRAM: IO19
:SETUP86
:ClLCD
:Disp " The SETUP86 program"
:Disp "has been called and"
:Disp "it restored all"
:Disp "default settings"
