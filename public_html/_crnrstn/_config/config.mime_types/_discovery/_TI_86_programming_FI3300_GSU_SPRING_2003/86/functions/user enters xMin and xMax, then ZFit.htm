Functions and Graphs - Lesson 11

This lesson will demonstrate how the ZFit instruction may be used to provide automatic scaling of the graph display window to ensure that the entire graph is contained within the viewing area.

In this program, the user enters a function to be graphed and then enters values for xMin and xMax. The program then uses the ZFit instruction to recalculate the values of yMin and yMax so that the entire graph of the function is contained within the graph window. This makes the most efficient use of the graph window when displaying the graph of the function.

New Instructions Introduced:

  • InpSt - This instruction allows the user to enter the function as a string and store it in the program.
  • St>Eq ( Y , y1 ) - This instruction converts the string variable Y into a function and stores it in the equation variable y1.
  • ZFit - This instruction recalculates yMin and yMax to include the minimum and maximum y values of the selected functions between the current xMin and xMax.

The InpSt instruction is found in the [ PRGM ], [ I / O ] menu.

The St>Eq instruction is found in the [ 2nd ], [ STRNG ] menu.

The ZFit instruction is found in the [ GRAPH ], [ ZOOM ] menu.

Note: The function must be entered in terms of  x. It is suggested that you use the [ x-VAR ] key to enter  x  in the function.

This example will demonstrate how the user can enter a function into a program and store the function in the equation variable y1. First, the function must be entered as a string and stored in the string variable Y using the InpSt instruction. Then, the St>Eq instruction is used to convert the string variable Y into a function and store it in the equation variable y1.

The user is then allowed to enter values for xMin and xMax. The program then uses the ZFit instruction to recalculate yMin and yMax as necessary to fit the entire graph of the function within the graph window.

Note: I suggest that you call the SETUP86 program to reset the calculator settings before running any program that involves graphing to insure that the program runs the way you are expecting it to. If you don't do this, you may be surprised by some unexpected results!

Tip: If you place a Pause instruction immediately following the DispG instruction, the program will pause until the user hits the [ ENTER ] key. This will allow time for the user to examine the graph before the bottom of the graph is covered up by the graph menu!

:SETUP86
:ClLCD
:Disp "ENTER THE FUNCTION"
:Disp "( USE x-VAR KEY FOR x )"
:InpSt "Y(x) =   ", Y
:St>Eq ( Y , y1 )
:ClLCD
:Disp "Enter  LOWER  and UPPER"
:Disp "values  for  x"
:Disp " "
:Prompt  xMin
:Prompt  xMax
:ClLCD
:Disp "Scaling  graph..."
:Disp "PLEASE  WAIT !"
:ZFit
:Pause

Note: In this example, ZFit recalculates yMin to be -15 and yMax to be 49
