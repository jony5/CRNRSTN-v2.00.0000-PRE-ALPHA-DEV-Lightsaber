Functions and Graphs - Lesson 17

This lesson will demonstrate how the fMin and fMax instructions may be used to find the minimum and maximum values of a function over a given interval.

 

New instructions introduced:

  • fMin - This instruction returns the value of  x  at which the minimum value of the function occurs on a given interval.
  • fMax - This instruction returns the value of  x  at which the maximum value of the function occurs on a given interval.

The format of the instructions are:

            fMin ( expression , x , lower , upper )

            fMax ( expression , x , lower , upper )

where:     expression  -  an expression or y function variable.
                               x   -  with respect to the variable x .
                        lower -  the left edge of the desired interval.
                        upper -  the right edge of the desired interval.

 

The fMin and fMax instructions are found in the [ 2nd ], [ CALC ] menu.

 

Note: The function must be entered in terms of  x. It is suggested that you use the [ x-VAR ] key to enter  x  in the function.

This example will demonstrate how the user can enter a function into a program and store the function in the equation variable y1. First, the function must be entered as a string and stored in the string variable Y using the InpSt instruction. Then, the St>Eq instruction is used to convert the string variable Y into a function and store it in the equation variable y1.

The user then enters values for xMin and xMax. The program then uses the fMin instruction to calculate the value of  x which yields the minimum value of the function between xMin and xMax and stores it in X1. It then uses the fMax instruction to calculate the value of  x  which yields the maximum value of the function between xMin and xMax and stores it in X2. Then, the program evaluates the function minimum at X1 and stores the result in MN. It evaluates the function maximum at X2 and stores the result in MX.

The program then displays the minimum and maximum values of the function and the values of x at which they occur.

Note: I suggest that you call the SETUP86 program to reset the calculator settings before running any program that involves graphing to insure that the program runs the way you are expecting it to. If you don't do this, you may be surprised by some unexpected results!

:SETUP86
:ClLCD
:Disp "ENTER THE FUNCTION"
:Disp "( USE x-VAR KEY FOR x )"
:InpSt "Y(x) =   ", Y
:St>Eq ( Y , y1 )
:ClLCD
:Disp "Enter  LOWER  and UPPER"
:Disp "values  for  x"
:Disp " "
:Prompt  xMin
:Prompt  xMax
:ClLCD
:Disp "Searching...."
:Disp "PLEASE  WAIT !"
:fMin ( y1 , x , xMin , xMax ) -->   X1
:fMax ( y1 , x , xMin , xMax ) -->   X2
: y1( X1 ) -->  MN
: y1( X2 )  -->  MX
:ClLCD
:Outpt ( 2 , 1 , "Y MIN = ")
:Outpt ( 2 , 9 , round( MN , 2 ))
:Outpt ( 3 , 1 , "WHEN x  = ")
:Outpt ( 3 , 10 , round( X1 , 2 ))
:Outpt ( 5 , 1 , "Y MAX = ")
:Outpt ( 5 , 9 , round( MX , 2 ))
:Outpt ( 6 , 1 , "WHEN x  = ")
:Outpt ( 6 , 10 , round( X2 , 2 ))
