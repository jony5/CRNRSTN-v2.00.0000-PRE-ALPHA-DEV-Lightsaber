Functions and Graphs - Lesson 3

This lesson is similar to the previous lesson except it will demonstrate a second method of evaluating a function which does not use the evalF instruction.

This lesson will show how the user can enter a function into a program and store that equation in the equation variable y1. The user can then enter a specific value for  "x", and evaluate the function at that value of  "x".

New Instructions Introduced:

  • InpSt - This instruction allows the user to enter the function as a string and store it in the program.
  • St>Eq ( Y , y1 ) - This instruction converts the string variable Y into a function and stores it in the equation variable y1.

The InpSt instruction is found in the [ PRGM ], [ I / O ] menu.

The St>Eq instruction is found in the [ 2nd ], [ STRNG ] menu.

Note: The function must be entered in terms of  x. It is suggested that you use the [ x-VAR ] key to enter  x  in the function.

This example will demonstrate how the user can enter a function into a program and store the function in the equation variable y1. First, the function must be entered as a string and stored in the string variable Y using the InpSt instruction. Then, the St>Eq instruction is used to convert the string variable Y into a function and store it in the equation variable y1.

The program will then allow the user to enter a value for "x" and store it in "A". It will then evaluate the y1 function when "x" is equal to "A" , by using y1( A ), and store the result in R. The program will then display the result.

:ClLCD
:Disp "ENTER THE FUNCTION"
:Disp "( USE x-VAR KEY FOR x )"
:InpSt "Y(x) =   ", Y
:St>Eq ( Y , y1 )
:Disp " "
:Disp "ENTER  x VALUE"
:Input " x = ", A
: y1( A )  -->  R
:ClLCD
:Outpt( 1 , 1 , "WHEN x =   ")
:Outpt( 1 , 10 , A )
:Outpt( 3 , 1 , "Y(x) =   ")
:Outpt( 3 , 7 , R )
     ENTER THE FUNCTION
    ( USE x-VAR KEY FOR x )
    Y(x) = 2 x  + 4

    ENTER  x  VALUE
    x  =  5
 
     WHEN   x  =  5

    Y(x) =  14
