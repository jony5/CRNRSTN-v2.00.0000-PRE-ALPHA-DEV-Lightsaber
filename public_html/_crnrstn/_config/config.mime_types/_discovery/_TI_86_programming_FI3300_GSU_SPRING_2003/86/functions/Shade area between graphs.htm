Functions and Graphs - Lesson 16

This lesson will show how the user can enter two functions into a program and then use the DispG instruction to display a graph of both functions, each with a different graph style. The Shade instruction will then be used to shade the area between the two graphs.

New instructions introduced:

  • Shade - This instruction shades the area between two functions. The shaded area is the area between the lower function and the upper function.

The format of the Shade instruction is:

            Shade ( lower function , upper function )

 

This example will demonstrate how the user can enter two functions into a program. The first function is stored in y1. The second function is stored in y2.

The program will then pause and allow the user to hit the [ ENTER ] key to view both graphs. The first function will graph as a normal line. The second function will graph as a thick line.

The program will then use the Shade instruction to shade the area between the lower and upper functions.

After the program ends, the graphs will remain on the screen and the graph menu will be made available for the user to choose options to explore the graph.

Note: I suggest that you call the SETUP86 program to reset the calculator settings before running any program that involves graphing to insure that the program runs the way you are expecting it to. If you don't do this, you may be surprised by some unexpected results!

:SETUP86
:ClLCD
:Disp "ENTER 1st FUNCTION"
:Disp "( USE x-VAR KEY FOR x )"
:InpSt "Y(x) =   ", Y
:St>Eq ( Y , y1 )
:ClLCD
:Disp "ENTER 2nd FUNCTION"
:Disp "( USE x-VAR KEY FOR x )"
:InpSt "Y(x) =   ", Y
:St>Eq ( Y , y2 )
:Outpt( 7 , 1 , "HIT ENTER TO ")
:Outpt( 8 , 1 , "VIEW GRAPH " )
:Pause
:ClLCD
:GrStl ( 1 , 1 )
:GrStl ( 2 , 2 )
:DispG
:Shade ( y1 , y2 )
   ENTER 1st FUNCTION
   ( USE x-VAR KEY FOR x )
   Y(x) = x^2  -  4
 
   ENTER 2nd FUNCTION
   ( USE x-VAR KEY FOR x )
   Y(x) = .5 x  + 2

  HIT  ENTER TO
   VIEW  GRAPH
