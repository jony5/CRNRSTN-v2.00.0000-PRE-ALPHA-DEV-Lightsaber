Functions and Graphs - Lesson 5

This lesson will demonstrate how the built-in constants stored in the TI-86 may be included in a Physics formula and then evaluated using input from the user.

A common problem in a Physics class is to use this formula to determine the gravitational force of attraction between two masses a given distance apart.

The formula for gravitational attraction is:   F  =  Gc * ( M1 * M2 ) / ( R^2 )

             Where :  F   =  force of attraction
                           Gc   =  gravitational constant
                           M1   = mass of the first object
                           M2   = mass of the second object
                           R   = Distance between the objects

The TI-86 contains 15 built-in constants which you may include in your programs. These constants are:

  • Na  -  Avogadro's number,  6.0221367E23
  • k  -  Boltzman's constant,   1.380658E-23
  • Cc  -  Coulomb constant,  8.9875517873682E9
  • ec  -  Electron charge,  1.60217733E-19
  • Rc  -  Gas constant,  8.31451
  • Gc  -  Gravitational constant,   6.67259E-11
  • g  -  Earth acceleration due to gravity,  9.80665
  • Me  -  Mass of an electron,   9.1093897E-31
  • Mp  -  Mass of a proton,  1.6726231E-27
  • Mn  -  Mass of a neutron,  1.6749286E-27
  • m0  -  Permeability in a vacuum,   1.2566370614359E-6
  • e0  -  Permittivity of a vacuum,   8.8541878176204E-12
  • h  -  Planck's constant, 6.6260755E-34
  • c  -  Speed of light in a vacuum,  299,792,458
  • u  -  Atomic mass unit,  1.6605402E-27
                              

Note: The built-in constants are found in the [ 2nd ], [ CONS ], [ BLTIN ] menu.

This example will first display the formula to the user, and then allow the user to enter values for  M1, M2, and R.

The program will then evaluate the formula and display the value of  F.

:ClLCD
:Disp "F = Gc * ( M1 * M2 ) / ( R^2) "
:Disp " "
:Prompt M1
:Prompt M2
:Prompt R
:Gc * ( M1 * M2 ) / ( R^2)   --> F
:Disp " "
:Disp "F = ",  F
