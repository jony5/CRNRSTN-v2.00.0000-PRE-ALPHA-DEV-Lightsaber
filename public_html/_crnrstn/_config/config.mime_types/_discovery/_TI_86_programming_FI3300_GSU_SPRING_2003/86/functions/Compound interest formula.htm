Functions and Graphs - Lesson 4

This lesson will demonstrate how a formula may be stored in a program and then evaluated using input from the user.

A common problem in an Algebra class is to use the formula for compound interest to determine the future value of money.

The formula for simple interest is:   F  =  P ( 1 + R / N ) ^ ( N * T )

             Where :  F   =  final amount
                           P   =  initial amount
                           R   = interest rate
                           N   = number of compounding periods
                           T   = Time ( in years )
                          

This example will first display the formula to the user, and then allow the user to enter values for P, R, N, and T. The program will then convert the interest rate to a decimal value by dividing by 100.

The program will then evaluate the formula and display the value of  F.

:ClLCD
:Disp "F = P ( 1 + R / N ) ^ ( N * T )"
:Disp " "
:Prompt P
:Prompt R
:Prompt N
:Prompt T
:R / 100 --> R
:P ( 1 + R / N ) ^ ( N * T ) --> F
:Outpt( 8 , 1 , "F  =   $ ")
:Outpt( 8 , 6 , round( F, 2 ) )
   F = P ( 1 + R / N ) ^ ( N * T )

    P=?  1000
    R=?  6
    N=?  12
    T=?  5

    F = $1348.85
