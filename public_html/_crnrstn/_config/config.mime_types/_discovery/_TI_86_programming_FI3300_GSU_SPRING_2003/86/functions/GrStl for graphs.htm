Functions and Graphs - Lesson 12

This lesson will demonstrate how any of the seven TI-86 graph styles may be included in a program by using the GrStl instruction. In this example, graph style number 4, which shades below the graph, will be demonstrated.

 

The format of the GrStl instruction is:

       GrStl ( y function # , graph style # )

where:.

  • y function # - You may select any y function by its number ( ex: for y1, use 1 )
  • graph style # - Enter the number of any of the seven available graph styles.

The seven graph styles are:

  1. normal line
  2. thick line
  3. shade above the line
  4. shade below the line
  5. animate the path of the line
  6. animate the path without showing the line
  7. dotted line

This example will demonstrate how the user can enter a function into a program and store the function in the equation variable y1. First, the function must be entered as a string and stored in the string variable Y using the InpSt instruction. Then, the St>Eq instruction is used to convert the string variable Y into a function and store it in the equation variable y1.

The program will then pause and allow the user to hit the [ ENTER ] key to view the graph. The y1 function will be graphed using graph style # 4, which will shade the area below the graph of the function. After the program ends, the graph will remain on the screen and the graph menu will be made available for the user to choose options to explore the graph.

Note: I suggest that you call the SETUP86 program to reset the calculator settings before running any program that involves graphing to insure that the program runs the way you are expecting it to. If you don't do this, you may be surprised by some unexpected results!

:SETUP86
:ClLCD
:Disp "ENTER THE FUNCTION"
:Disp "( USE x-VAR KEY FOR x )"
:InpSt "Y(x) =   ", Y
:St>Eq ( Y , y1 )
:Outpt( 7 , 1 , "HIT ENTER TO ")
:Outpt( 8 , 1 , "VIEW GRAPH " )
:Pause
:ClLCD
:GrStl ( 1 , 4 )
:DispG
:Pause
   ENTER THE FUNCTION
   ( USE x-VAR KEY FOR x )
   Y(x) = x^2  -  2


  HIT  ENTER TO
   VIEW  GRAPH
